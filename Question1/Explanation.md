## Command: id

The `id` command displays the current user's UID, GID, and the groups associated with the account. It verifies the user's identity and permissions on the system.

## Command: echo $SHELL

The `echo $SHELL` command displays the default shell assigned to the current user. It confirms which command interpreter is configured for the session.

## Command: pwd

The `pwd` command prints the absolute path of the current working directory. It helps verify the user's current location in the filesystem.

## Command: ls -la

The `ls -la` command lists all files and directories, including hidden files, with detailed information such as permissions, ownership, and file size.

## Command: ping -c 4 google.com

The `ping` command sends ICMP packets to a remote host to test network connectivity. Successful replies indicate that the system has internet access.
