## Creating the Workspace
I created a structured project workspace containing separate directories for documentation, source code, and backups.

## Command: find .
The `find .` command displays the complete directory structure of the current workspace. It confirms that the required folders and files were created successfully.

## Checking Initial Permissions
The `ls -l` command displayed the initial permissions and ownership of the files and directories.

## Modifying Permissions
The `chmod` command changed the access rights to improve security by restricting unauthorized access.

## Checking Ownership
The ownership information confirms which user owns each file and directory.

## Viewing Umask
The `umask` command displays the default permission mask that affects permissions of newly created files.
